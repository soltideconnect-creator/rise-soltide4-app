## ğŸ”§ Troubleshooting

### Notifications Not Working
**Problem**: Not receiving daily reminders  
**Solutions**:
- Check browser notification permissions
- Ensure notifications enabled for the site
- Try granting permission again
- Check device notification settings

### Data Not Saving
**Problem**: Habits disappear after closing browser  
**Solutions**:
- Ensure not in private/incognito mode
- Check if localStorage is enabled
- Verify sufficient storage space
- Try a different browser

### Performance Issues
**Problem**: App feels slow or laggy  
**Solutions**:
- Clear browser cache
- Close unnecessary tabs
- Update browser to latest version
- Restart browser

### Display Issues
**Problem**: Layout looks broken  
**Solutions**:
- Refresh the page (Ctrl+R or Cmd+R)
- Clear cache and hard reload (Ctrl+Shift+R)
- Try a different browser
- Check screen zoom level (should be 100%)
