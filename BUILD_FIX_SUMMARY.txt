Netlify build was failing with duplicate symbol errors:
- "The symbol 'DEBUG_MODE' has already been declared"
- "The symbol 'debugLog' has already been declared"

ROOT CAUSE:
-----------
Multiple files (Stats.tsx, paystack.ts, googlePlayBilling.ts) were each
declaring their own DEBUG_MODE, debugLog, and debugError constants.
The esbuild bundler detected these as duplicate symbols during production build.

SOLUTION:

all files to import from it instead of declaring their own debug functions.
